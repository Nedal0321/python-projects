#How To Use

1. Clone or download this repository to your local machine.
2. Make sure you have Python installed on your machine.
3. Run the script using the following command:

# python downloader.py
	
4. Enter the URL of the file you want to download.
5. The script will download the file, calculate its hash value, and save both the downloaded file and its hash value to separate files in the current directory.

Requirements:
- Python 3.x
- requests library (install using #pip install requests)
